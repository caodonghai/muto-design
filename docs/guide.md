# 使用指南

## 安装

:::code-group

```bash [npm]
npm install -D muto-design
```

```bash [yarn]
yarn add -D muto-design
```

```bash [pnpm]
pnpm add -D muto-design
```

:::

## 使用

```tsx | pure
import { WaterMark } from 'muto-design';

export default () => {
  return <WaterMark text="example@xx.xx.com"/>
}
```
